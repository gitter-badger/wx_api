#微信SDK
####Get Start
1. 引入包
2. ````classpath````下加入````wx.xml````配置  
    在其中加入：
    ````
    <?xml version="1.0" encoding="UTF-8"?>
    <configuration>
        <listeners>
            <listener>com.piggsoft.listener.TextWXEventListenerTest</listener>
        </listeners>
    </configuration>
    ````
